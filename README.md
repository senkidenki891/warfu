# warfu

Каркас проекта на **TypeScript + React (Vite-style)** с разделением UI и боевой симуляции.

## Структура

- `src/ui` — компоненты интерфейса и панели;
- `src/sim` — чистая логика симуляции, контроллер и селекторы;
- `src/data` — конфиги и справочники;
- `src/scenario` — определения сценариев;
- `src/tests` — unit-тесты.

## Архитектурный контракт

1. UI (`src/ui`) читает данные только через селекторы/контроллер из `src/sim`.
2. UI не содержит боевой логики и не вычисляет правила симуляции.
3. Изменения состояния происходят через API движка (`src/sim/engine.ts`).

## Команды

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run test`
- `npm run format`
